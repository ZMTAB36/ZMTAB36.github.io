1,权限开启-允许

2，点击开发者局域网IP

3，电脑端浏览器输入IP地址加端口号

4，没有电脑手机端浏览器输入IP也可以

5，开始构建，工程名随便起名

6,手机端构建工程的时候手机开启旋转屏幕

7，点击_init_.py清空内容

复制下面代码粘贴，保存在打开短视频，点击运行

简单写好的刷极速版短视频，随便哪个短视频平台都可以

#导入包

from airscript.action import slide #滑动

import time #停留时间包

import random  #随机值包

#循环

while True:

#滑动坐标

      slide(471,1821,484,396,683)

      t = random.randrange(3,15) #随机数，秒

      time.sleep(t)  #暂停几秒

      print("停留",t,"秒")
