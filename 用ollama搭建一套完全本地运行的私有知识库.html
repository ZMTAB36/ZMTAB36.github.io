用ollama搭建一套完全本地运行的私有知识库

阅读量：1万+

RAG检索增强生成，让大模型基于你自己的文档来回答问题。

本地运行的私有知识库，数据不出你的电脑，用到的工具全部免费开源。

你上传文档（PDF、Word、TXT 等）。

系统把文档切成小块，转成向量存起来。

你提问时，系统先从向量库里找到最相关的段落。

把这些段落作为上下文，交给大模型生成回答。

使用的工具AnythingLLM推荐新手首选。

有独立桌面应用（Windows/Mac/Linux），下载就能用。

内置 RAG + AI Agent + 向量数据库，不用单独装一堆组件。

支持 30+ 种 LLM 接入，包括 Ollama 本地模型。

支持 PDF、Word、TXT、网页等多种格式。

界面友好，拖拽上传文档就行。

本地电脑安装ollama，先下载一个AI问答模型配合。

在安装索检模型：

ollama pull nomic-embed-text

去官网 anythingllm.com下载桌面版安装包。

在AnythingLLM的设置里：

LLM 设置 → 选择 Ollama → 模型选 gemma4:e2b或者e4b，根据电脑配置选择。

嵌入模型 → 选择 Ollama → 模型选 nomic-embed-text

向量数据库 → 保持默认的 LanceDB 就行（内置的，不用额外安装）

Ollama 默认运行在 http://localhost:11434，AnythingLLM 会自动检测到。

点击左侧 "New Workspace" 创建一个工作区，比如叫"产品文档"

点击工作区设置里的Upload Documents

把你的文档拖进去——支持 PDF、Word、TXT、Markdown、网页链接等

点击Move to Workspace开始向量化处理

处理完成后，你就可以在聊天框里提问了！

在工作区设置里开启 "Query Mode"，模型只会基于文档内容回答。

不会用自己的知识"补充"适合对准确性要求高的场景。
