MicroPython-ESP32文档-引脚和GPIO

from machine import Pin

p0 = Pin(0, Pin.OUT) # Pin.OUT输出

p0.on() # set pin to "on" (high) level

p0.off() # set pin to "off" (low) level

p0.value(1) # 高电平1，低电平0

p2 = Pin(2, Pin.IN) # create input pin on GPIO2

print(p2.value()) # get value, 0 or 1

p4 = Pin(4, Pin.IN, Pin.PULL_UP) # enable internal pull-up resistor

p5 = Pin(5, Pin.OUT, value=1) # set pin high on creation

p6 = Pin(6, Pin.OUT, drive=Pin.DRIVE_3) # DRIVE_3电流为40毫安

支持四种驱动强度，使用构造函数或方法的关键字参数，具有不同的对应 安全的最大拉电流/灌电流和近似内部驱动器电阻：drivePin()Pin.init()

Pin.DRIVE_0： 5mA / 130欧姆

Pin.DRIVE_1： 10mA / 60欧姆

Pin.DRIVE_2：20mA / 30 欧姆（如果未配置，则为默认强度）

Pin.DRIVE_3： 40mA / 15欧姆
